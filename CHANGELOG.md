# Changelog

## [2.1.0] — 2026-06-12

### Added
- **New file** — File > New 或 Ctrl+N 创建空白文件
- **Close file** — File > Close 关闭当前文件，清空编辑器
- **未保存追踪** — 编辑器内容变化时，标题栏和状态栏显示 `●` 标记
- **未保存提示** — 关闭窗口、打开/新建/关闭文件时，若内容已修改则提示保存或放弃
- **同步滚动** — 左侧编辑器滚动时，右侧预览区按比例跟随，以首行为基准对齐

### Changed
- **File 菜单** — 重构文件操作，整合 New、Close 菜单项及快捷键
- **保存行为** — 保存操作后清除未保存标记
- **窗口关闭流程** — Exit 和 Alt+F4 均检查未保存内容，防止数据丢失

### Fixed
- 双击 `.md` 文件打开已有窗口时，先检查当前文件是否已修改

---

## [2.0.0] — 2026-06-12

### Added
- 初始版本：基于 Tauri 2.x 重写，替代原有 Python + tkinter 版本
- 分屏编辑：左侧源码编辑，右侧实时 HTML 预览（marked.js 渲染）
- 工具栏快捷插入：标题、粗体、斜体、删除线、引用、列表、链接、图片、代码、表格、分割线
- 4 套主题：Light / Dark / Solarized Light / Solarized Dark
- 缩放显示：50%~200%，Ctrl+= / Ctrl+- / Ctrl+0，缩放比例自动记忆
- 记忆上次文件：关闭后重新打开，自动恢复上次编辑的文件
- PDF 导出：基于 WebView2 PrintToPdf，支持中文
- 状态栏：文件名、行列号、当前行元素类型、缩放比例
- Windows 右键关联
- 单实例运行
- 便携配置：主题、窗口状态、缩放、最近目录保存到系统配置目录
